***ICBM:Escalation 单位数据生成程序***
-----
如你所见，这是一个用于生成《ICBM:Escalation》单位数据的一个简易的Py TTK程序，为了LRP模组而创建。这是我接触Python的后的第一个Demo，大部分内容都是AI写的。
  
功能：
1. 填写游戏已经定义的参数和值；
2. 自定义雷达、武器配置
3. 创建“Modifier”、“Improved By”等行；
4. 允许自定义其他参数和值。
5. 通过JSON保存目前的填写项，以便随时存取；
6. 生成自定义单位的代码；
